# Data Center Relationships Visualizer

Interactive React prototype for visualizing data center facility, power, and thermal relationships.

## Run Locally

```powershell
npm install
npm run dev
```

Vite will print the local URL, usually `http://localhost:5173/`.

## Project Layout

- `src/` - React application source, graph rendering, data definitions, and layout logic.
- `public/object-images/` - transparent PNG equipment illustrations used inside graph nodes.
- `scripts/` - utility scripts, including object image generation.
- `docs/` - implementation plan, asset plan, original spec, and thermal update plan.
- `qa/` - local visual QA screenshots captured during development.
